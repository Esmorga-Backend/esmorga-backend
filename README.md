## Description
Backend of Esmorga.An app to manage events.

## Installation

First clone de repository:
```bash
$ git clone https://github.com/Esmorga-Backend/esmorga-backend.git
```
And then execute:

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
